
Getting Started:
   1) Clone the repository to root folder of server
   2) Create two hosts with current names:
        -'https://front.shpp' - root folder is 'front'
        -'https://back.shpp' - root folder is 'back'
   3) Create DB with name 'todo' (all requested tables are creating automaticlly);
   4) In depense of version which you want to use (v2, v3),
      choose version in 'index.php' file situated at the 'front' folder;
